# QR Studio

A modern web-based QR code generator with advanced customization options.

## Features

- **Multiple Content Types**: URL, Plain Text, Email, Phone, Wi-Fi, vCard
- **Customizable Styling**: Square, Round, Diamond dot patterns
- **Corner Frames**: Square, Rounded, Circle styles
- **Color Options**: Custom colors with gradient support (Horizontal, Vertical, Diagonal, Radial)
- **Color Presets**: Quick-start color schemes
- **Contrast Indicator**: Real-time contrast ratio display
- **Transparent Background**: For overlay use
- **Logo Support**: Upload and center custom logos
- **Call-to-Action Text**: Add text below QR code
- **Live Preview**: Real-time updates as you type
- **Export Formats**: PNG, SVG, JPEG
- **Print-Ready**: Up to 4096px with DPI warning
- **Save/Load Presets**: Export and import settings

## Usage

Open `web/index.html` in a browser. No server required.
